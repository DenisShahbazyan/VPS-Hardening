# VPS Hardening Manager

Интерактивный CLI-инструмент для быстрого и безопасного hardening Ubuntu/Debian VPS.
Меню-интерфейс на русском и английском языках. Все изменения конфигов делаются с автоматическим бэкапом. Поддерживается полный откат при ошибке.

---

## Запуск

```bash
wget -qO ~/harden.sh https://raw.githubusercontent.com/DenisShahbazyan/VPS-Hardening/master/harden-manager.sh && sudo bash ~/harden.sh
```

```

║           VPS Hardening Manager              ║
║                  Ubuntu                      ║
╚══════════════════════════════════════════════╝

  Статус системы
  ┌─────────────────────────────────────────┐
  │ Пользователи   alice (1 ключ), bob      │
  │ SSH порт       39217                    │
  │ UFW            активен                  │
  │ CrowdSec       активен                  │
  │   └─ cs-firewall-bouncer  активен       │
  │ Root SSH       отключён                 │
  └─────────────────────────────────────────┘

  1) Управление пользователями
  2) Управление SSH ключами
  3) Сменить порт SSH
  4) Настроить UFW
  5) Настроить CrowdSec
  6) Отключить root-логин SSH

  7) Автонастройка (всё сразу)
  8) Вернуть по умолчанию

  Enter) Выход
```

### Пункты меню

| #   | Действие                  | Описание                                             |
| --- | ------------------------- | ---------------------------------------------------- |
| 1   | Управление пользователями | Создать или удалить системного пользователя с sudo   |
| 2   | Управление SSH ключами    | Добавить или удалить публичный ключ для пользователя |
| 3   | Сменить порт SSH          | Поменять порт sshd; UFW обновляется автоматически    |
| 4   | Настроить UFW             | Включить/отключить файрвол со стандартными правилами |
| 5   | Настроить CrowdSec        | Установить/удалить CrowdSec и bouncers               |
| 6   | Root SSH                  | Включить или отключить `PermitRootLogin`             |
| 7   | Автонастройка             | Запустить все шаги разом с откатом при ошибке        |
| 8   | Сброс                     | Вернуть сервер к исходным настройкам                 |

---

## Автонастройка (пункт 7)

Выполняет 7 шагов последовательно. При ошибке на любом шаге — полный автоматический откат:

1. Создать пользователя и добавить его в `sudo`
2. Добавить SSH ключ
3. Установить случайный пароль для `sudo` + отключить вход по паролю SSH
4. Выбрать случайный SSH порт (10 000 – 65 000)
5. Настроить UFW
6. Установить CrowdSec + `crowdsec-firewall-bouncer-iptables`
7. Отключить root-логин SSH

По завершении выводит итоговую сводку с паролем и командой для подключения:

```
ssh -p <порт> <пользователь>@<IP>
```
